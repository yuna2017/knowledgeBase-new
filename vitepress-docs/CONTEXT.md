# 内容术语与维护规范

面向燕大师生的在线生活指南。内容通过 tags 组织，分类从旧版固定栏目迁移为标签化。

## Language

**tag**:
文章的扁平化分类标识。一篇文章可拥有多个 tag，读者通过 tag 聚合或检索来发现内容。旧版固定栏目已废弃，全部转为 tag。
_Avoid_: 分类、栏目、板块

**校园服务门户 / 一网通办**:
学校统一的在线服务入口 (ehall.ysu.edu.cn)，用于登录教务、图书馆、缴费等。
_Avoid_: 在线服务、服务大厅

**校园网接入**:
连接校园网并进行认证的过程，包括宿舍有线/无线、教学区 Wi-Fi、运营商宽带及校外 VPN 接入。
_Avoid_: 网络、校园网络

**免费资源**:
利用学生身份或免费额度获取的服务与资源，包括学生专属权益（如 GitHub Student Pack）和面向大众的免费层/免费额度（如 Cloudflare 免费层）。
_Avoid_: 过于口语化的免费称呼、学生优惠（单独使用）

**作者 / authors**:
对当前文章有实质贡献的个人，包括初稿撰写者和后续修改、补充、核实的贡献者。文章按 tags 分类，作者名只用于归属记录，不参与主要分类。
_Avoid_: 维护者、所有者

## 时效性与地区适用性

- 校园政策、套餐和系统流程以燕山大学当期官方通知为准。登录后页面或学生实测需要注明日期。
- 免费额度、价格、模型、软件版本和学生权益必须引用官方页面，并标注最后核验时间。
- 海外服务不能只写“可以访问”。还要检查中国大陆是否属于官方支持地区，以及账号、支付、数据处理和服务条款限制。
- 对外公开网站时，区分境外托管与中国大陆服务器；后者通常涉及 ICP 备案等要求。
- 不推荐来源不明的镜像、中转站和下载站，也不把第三方服务预置项视作知识库背书。
