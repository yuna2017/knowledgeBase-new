# MOOC 与免费学习资源

## 国内平台

- **[中国大学 MOOC](https://www.icourse163.org/)**：国内高校课程较多，免费学习范围、作业和证书规则由当期开课设置决定
- **[学堂在线](https://www.xuetangx.com/)**：清华系平台，计算机与理工科课程质量高
- **B 站**：高校、教师和课程官方账号会发布公开课。优先使用明确获得授权的官方内容，不推荐来源不明的“搬运版”

## 海外平台

- **[Coursera](https://www.coursera.org/)**：部分课程提供免费预览、旁听或助学金，是否开放以及包含哪些内容，以具体课程页面为准
- **[edX](https://www.edx.org/)**：部分课程提供免费 Audit 方式，测验、作业或证书可能需要付费
- **[MIT OpenCourseWare](https://ocw.mit.edu/)**：MIT 课程资料完全公开，无需注册
- **[freeCodeCamp](https://www.freecodecamp.org/chinese/)**：编程实战路线 + 免费证书，有中文版

## 经典自学路线参考

- **计算机科学全套**：[CS自学指南（csdiy.wiki）](https://csdiy.wiki/) —— 中文社区维护的名校课程自学地图（MIT 6.006、CS61 系列、CMU 15-213 等），含资料和作业获取方式
- **[The Missing Semester](https://missing-semester-cn.github.io/)**：MIT 的工具课，内容包括 Shell、Git、调试、数据处理（与本板块 [Git 入门](./tech-git-github.md)互补）

## 使用建议

1. **先试听再决定**：如果课程提供免费预览或旁听，先确认难度、语言和作业量，再考虑是否付费。
2. **证书理性看待**：MOOC 证书对保研 / 求职是锦上添花，学到的东西本身才是主要收益。
3. Coursera、edX、MIT OCW 等境外网站在中国大陆的连接、视频加载、登录验证和支付可能不稳定；校园网不等于国际专线，不能保证解决这些问题。
4. 只使用课程官方提供的下载或离线功能。不要在来源不明的网站输入学校账户、支付信息或课程平台密码。
