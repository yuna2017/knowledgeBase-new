# GitHub 学生包与教育优惠

## GitHub Student Developer Pack

GitHub Student Developer Pack 把 GitHub 自身权益和合作商优惠放在同一页面。通过学生认证后，还要按各合作商的要求单独领取；不是页面上的每项服务都会自动开通。

**目前值得关注的项目：**

- **GitHub Pro 和 Codespaces**：认证学生可获得相应权益，但包含量会随计划调整。以 [GitHub Education 学生说明](https://docs.github.com/en/education/about-github-education/github-education-for-students/about-github-education-for-students)和账户账单页显示为准。
- **GitHub Copilot**：学生计划及其开放状态正在变化。核验时，官方页面提示新的学生计划注册暂时暂停，因此不要照旧教程假定可以立即领取。
- **JetBrains**：学生授权需要在 JetBrains 单独申请和续期。详细流程见[校园邮箱板块](./campus-mail-index.md)。
- **DigitalOcean**：学生包页面仍可能提供新用户优惠，但有领取期限、使用期限和不适用服务。领取前阅读优惠卡片中的完整条款，不在这里写死金额。
- **域名**：当前可核验到 `.TECH` 和 Name.com 的部分域名权益；可选后缀、免费期和续费价格以学生包页面为准。旧教程中的 Namecheap 免费 `.me` 已不适用。
- **Termius**：当前学生权益按学生身份有效期提供，不是固定“两年”。

**申请要点：**

- 入口：[GitHub Student Developer Pack](https://education.github.com/pack) 或 GitHub 账户中的 Education benefits 页面。
- 准备学校邮箱和能证明当前在读状态的材料，姓名等信息按证件和页面要求填写。GitHub 可能根据学校和账户情况要求补充材料。
- 提交方式、是否调用摄像头、审核时间和认证到期日都由当前页面及审核结果决定，没有统一的固定天数。
- 建议先开启两步验证，并保存恢复码。中国大陆手机号接收短信可能不稳定，优先考虑通行密钥或身份验证器生成的一次性验证码。

## 学生包之外的教育优惠

| 服务 | 权益 | 认证方式 |
|------|------|----------|
| **Notion Education** | 符合条件的学生可申请教育计划，具体功能看官方计划页 | 学校邮箱或官方要求的其他材料 |
| **Figma for Education** | 独立于 GitHub 学生包的教育计划 | 按 Figma 当前流程验证学生身份 |
| **Microsoft 365 Education** | 是否可用以及包含哪些应用，由学校租户和管理员分配决定 | 查看学校正版化平台和本人学校账户 |
| **Azure for Students** | 面向符合条件学生的全球版 Azure 优惠，额度和可用服务看实时页面 | 按 Azure 当前流程验证学生身份 |

## 中国大陆使用前先确认

- GitHub 学生包只负责学生认证，不保证合作商正式支持中国大陆。注册地区、手机号、身份验证、国际支付和服务条款要分别核对。
- Azure for Students 指全球版 Azure，不是世纪互联运营的 Azure 中国。两者账户、区域和计费体系不同，不能混用。
- 境外服务即使能打开，也可能出现下载慢、验证失败或支付受限。不要为了领取优惠提交虚假地区或身份信息。
- Cursor、Gemini 等限时学生活动已经出现过停止新申请或地区受限的情况。遇到“免费一年”之类旧教程，先打开产品官方教育页面确认活动仍在进行。
