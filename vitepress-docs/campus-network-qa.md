# 校园网常见问题

## 连接后没有弹出认证页

先确认设备已经连接学校有线网络或校园 Wi-Fi，再用浏览器打开 [auth1.ysu.edu.cn](http://auth1.ysu.edu.cn)。

如果浏览器显示的是旧页面或空白页，可关闭页面后重新连接网络，再重新打开。具体认证步骤见[校园网连接与认证](./campus-network-connect.md)。

## 认证页能打开，但登录失败

先核对账号、密码和服务选项，并完整记录页面上的错误信息。服务名称会随本人账号状态和已办理业务变化，以当前页面为准。

如果统一身份账号也无法登录一网通办，应先按学校当前的账号找回流程处理。只有运营商服务登录失败时，联系对应运营商的官方客服或营业厅，不要把密码或验证码交给个人推销者。

## 显示认证成功，但仍然无法上网

可以按以下顺序排查：

1. 断开校园网后重新连接，再打开一个普通网页测试。
2. 重新进入认证页，确认当前会话是否仍然在线。
3. 登录自助服务，检查本人账号状态、当前服务和在线设备。页面具体栏目以实际显示为准。
4. 用另一台设备或另一处校园网接入点测试，判断问题来自单台设备还是当前网络区域。

仍无法解决时，保留发生时间、所在楼宇或房间、设备类型和页面报错，按认证页或一网通办当前公布的校园网络支持渠道反馈。

## 套餐、流量和设备数是多少

截至2026年新生入学，校园网服务每月每人拥有**60G**免费流量，不可结转至下月，最大同时登录设备数为**3台**；中国移动、中国电信、中国联通等运营商服务，每人每月**不限**流量，最大同时登录设备数为**1台**；校园网对于[B站](https://www.bilibili.com/)与[魔搭社区](https://www.modelscope.cn/)实行免流量政策。

## 校外打不开图书馆或教务资源

学校只向校内开放的部分资源，可以尝试从 [WebVPN（校外访问系统）](./campus-network-vpn.md)进入。是否能访问某个数据库或系统，取决于学校当前采购范围和 WebVPN 发布列表。

## 安全提醒

- 不要共享统一身份账号、校园网密码和短信验证码。
- 不要安装来源不明的认证客户端、证书或所谓加速工具。
- 学校原则上禁止私接路由器。
