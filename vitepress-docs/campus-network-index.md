# 校园网简介

在宿舍、教学楼或图书馆使用学校有线网络和校园 Wi-Fi 时，通常需要通过校园网认证系统接入。校园网可用于一般互联网访问，也可用于访问部分仅向校内开放的资源。

学校购买的数据库范围、软件授权和网络开放策略可能变化，能否访问应以图书馆、软件平台和当前认证系统为准。

## 如何接入

校园有线网络和 Wi-Fi 的具体名称、覆盖范围以现场标识和学校当前通知为准。认证入口与操作方法见[校园网连接与认证](./campus-network-connect.md)。

## 校外访问学校资源

部分校内资源在校外不能直接打开，可尝试通过学校的 [WebVPN（校外访问系统）](./campus-network-vpn.md)访问。WebVPN 只提供学校已发布的资源入口，不是通用互联网代理。

## 要不要办理校园流量卡（校园卡）业务？

> **提示**
>
> 本文不建议或反对办理任何运营商套餐。是否办理，应根据自己的流量需求、预算和合同条件决定。

截至2026年新生入学，校园网服务每月每人拥有**60G**免费流量，不可结转至下月，最大同时登录设备数为**3台**；中国移动、中国电信、中国联通等运营商服务，每人每月**不限**流量，最大同时登录设备数为**1台**；校园网对于[B站](https://www.bilibili.com/)与[魔搭社区](https://www.modelscope.cn/)实行免流量政策。

运营商校园卡和宽带属于商业套餐，资费、流量范围、宽带权益、设备限制、优惠期限及解约条件会因运营商、入网年份和办理批次而变化。办理前应通过运营商官方 App、客服或营业厅逐项确认：

- 月租、预存款和优惠结束后的价格；
- 国内通用流量与定向或省内流量的区别；
- 是否包含校园宽带、可同时在线的设备数；
- 合约期限、销户方式和提前解约费用。

> **注意**
>
> 不要仅凭个人推销者的口头承诺办理，也不要向他人提供账号密码或短信验证码。所有费用和权益以运营商正式订单、合同及官方渠道查询结果为准。
